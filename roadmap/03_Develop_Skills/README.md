# 03 Develop Skills

After fundamentals, expand your skill set with practical and interview-focused abilities.

## Key Skills
- Data Structures & Algorithms
- Git & GitHub
- Problem Solving
- Aptitude & Reasoning

## Suggested Actions
- Solve DSA problems daily using online judges such as LeetCode or CodeChef.
- Create Git repositories, practice branching, and publish projects on GitHub.
- Practice logical reasoning and aptitude problems from common placement materials.
- Track progress with a study schedule and review mistakes regularly.

## Outcome
- Strong coding problem-solving skills
- Version control fluency
- Improved interview readiness for technical assessments
