# 04 Build Projects

Build projects that demonstrate your skills and become part of your portfolio.

## Project Plan
- Mini Project 1: Solve a small practical problem with a complete implementation.
- Mini Project 2: Build a second small app or automation tool.
- Major Project: Create a larger, portfolio-grade project with real-world scope.
- Portfolio Website: Showcase your projects, skills, and contact details.

## Suggested Actions
- Choose projects aligned with your chosen career domain.
- Write clean code, add documentation, and include screenshots or demo links.
- Deploy or publish projects when possible.
- Build a simple portfolio site using HTML/CSS/JavaScript or a static site generator.

## Outcome
- Three completed projects with documented repositories
- Portfolio website that highlights your best work
- Demonstrable evidence of practical ability
