# 01 Career Domain Selection

For an unemployed student, choosing the right career domain sets the foundation for the roadmap.

## Recommended Domains
- Web Development
- Data Science
- AI/ML
- Cyber Security
- Cloud Computing

## What to do
- Research each domain and identify what interests you most.
- Compare domain roles, salary estimates, required tools, and learning paths.
- Select one domain to focus on for the next 3–6 months.
- If unsure, start with a broad domain like Web Development or Data Science.

## Outcome
- Chosen target domain
- Clear shortlist of roles to pursue
- Personal motivation statement for the selected career path
