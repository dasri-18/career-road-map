# 05 Create Professional Profiles

Build the professional presence needed to attract internship opportunities.

## Profiles to Create
- Resume
- LinkedIn Profile
- GitHub Profile

## Suggested Actions
- Write a clear resume with your objective, education, projects, skills, and contact details.
- Optimize your LinkedIn profile with a strong headline, summary, and project experience.
- Keep your GitHub profile active with polished repositories and README documentation.
- Add links to your portfolio and GitHub on your resume and LinkedIn.

## Outcome
- Internship-ready resume
- Professional LinkedIn presence
- GitHub profile that showcases your projects
