# 02 Learn Fundamentals

Build a strong computer science foundation before moving to advanced skills.

## Core Topics
- Programming: Python / Java / C++
- DBMS: SQL, normalization, queries, transactions
- OOPs: classes, inheritance, polymorphism, encapsulation
- Operating Systems: processes, threads, memory management
- Computer Networks: OSI model, TCP/IP, routing, protocols

## Suggested Actions
- Complete beginner courses or tutorials for chosen programming languages.
- Practice SQL queries and design one small database schema.
- Implement OOP concepts with simple examples and mini projects.
- Read introductory OS and Networking material for software engineering interviews.

## Outcome
- Clear understanding of key CS fundamentals
- Basic programming and database skills
- Preparedness for technical interview topics
