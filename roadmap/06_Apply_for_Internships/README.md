# 06 Apply for Internships

Target internships at leading companies while applying to additional firms for better odds.

## Target Companies
- Google
- Microsoft
- Amazon
- Infosys
- TCS
- Wipro
- Accenture

## Suggested Actions
- Search official internship openings on each company’s careers page.
- Prepare tailored applications and submit resumes with a strong cover letter.
- Use LinkedIn, job boards, and campus placement portals.
- Track applications, deadlines, and follow-up status.

## Outcome
- Multiple internship applications submitted
- Company-specific application tracking
- Better visibility into interview timelines
