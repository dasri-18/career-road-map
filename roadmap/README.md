# Career Roadmap

This roadmap is designed for an unemployed student who wants to build a strong internship-ready career path.

Unemployed Student

│   ├── Web Development
│   ├── Data Science

│   ├── Cyber Security
│   └── Cloud Computing

├── Learn Fundamentals

│   ├── DBMS

│   ├── Operating Systems
│   └── Computer Networks
│
├── Develop Skills
│   ├── Data Structures & Algorithms
│   ├── Git & GitHub
│   ├── Problem Solving
│   └── Aptitude & Reasoning
│
├── Build Projects
│   ├── Mini Project 1
│   ├── Mini Project 2

├── Apply for Internships

│   └── Communication Skills
│
└── Get Internship
    ├── Gain Experience
    ├── Learn Industry Tools
    └── Convert Internship to Full-Time Job

## Purpose

This tree is a structured learning plan to help an unemployed student go from choosing a career domain to landing and converting an internship.

Each folder under `roadmap/` contains more details for the stage and actions to complete.
